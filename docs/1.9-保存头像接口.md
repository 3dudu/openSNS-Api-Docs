**简要描述：** 

- 保存头像接口

**请求URL：** 
- ` http://172.16.3.72:99/index.php/core/file/saveavatar.html`
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|crop |是  |string |头像裁剪尺寸数据|
|path |是  |string |服务端头像路径 ： ./Uploads/Avatar/1/596451a6a8b52.jpg  |

 **返回示例**

``` json
{
  "data": null,
  "info": "头像更新成功！",
  "status": 1
}
```
