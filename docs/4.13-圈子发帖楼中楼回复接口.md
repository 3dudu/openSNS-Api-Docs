**简要描述：** 

- 圈子发帖楼中楼回复接口

**请求URL：** 
- ` http://172.16.3.72:99/index.php/api/group/dosendlzlreply`
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|content |是  |int |回复内容   |
|to_f_reply_id |是  |int |主楼回复id   |
|to_reply_id |否  |int |楼中楼回复时必传, 表示被回复子楼id   |

 **返回示例**

``` 
{
  "data": 18,
  "html": false,
  "status": 1,
  "info": "回复成功！【积分：+1分】【积分：+1分】"
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|data |int |本次回复id   |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
